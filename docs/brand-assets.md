# Recursos de marca

`public/brand/` es la única fuente de recursos de marca servidos por StakeControl.

## Variantes

- `stakecontrol-logo-horizontal.svg`: logotipo principal para navbar, landing y documentos sobre fondos claros.
- `stakecontrol-logo-white.svg`: logotipo para fondos oscuros.
- `stakecontrol-logo-monochrome-dark.svg`: variante de una tinta para impresión o soportes monocromos.
- `stakecontrol-icon.svg`: isotipo transparente para sidebar o espacios compactos.
- `favicon.svg`: favicon escalable.
- `stakecontrol-app-icon-1024.png`: icono de aplicación/PWA.

## Uso

- Navbar de escritorio: 190–240 px de ancho.
- Navbar móvil: 150–180 px de ancho.
- Isotipo en sidebar: 32–40 px.
- Mantener una zona de seguridad equivalente al ancho del trazo exterior del escudo.
- No deformar el logotipo ni recolorearlo fuera de la paleta.

## Paleta

- Azul control: `#1E3A8A`
- Cian analítico: `#0891B2`
- Azul noche: `#0F172A`
- Blanco: `#FFFFFF`

## Tipografía

Usar Inter como tipografía base:

```css
font-family: "Inter", system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif;
```
